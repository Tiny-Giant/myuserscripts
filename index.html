About

This repository is where I store the user scripts that I have written for Stack Overflow. They aren't all polished, and most are incomplete. It should take about 6-8 weeks to finish them all. Bug reports are appreciated, feature requests are useful but don't expect anything to happen right away.

Debugging

AJAX Listener

A simple AJAX listener that intercepts all AJAX requests made on a page and logs them to the console for review.

Enable Logging

Enables Stack Overflow's built in logging / debugging system.

Chat

Moderation Request Archiver

Provides an interface to archive formatted moderation requests in a given chat room. Must be a room owner or moderator of the current room for the interface to show up. This is an old version, the newest version is being maintained by the SOCVR More information to come.

Moderation Request Generator

Provides an interface to automate the formatting and sending of moderation requests from chat to a given chat room. Currently only works for close vote requests. More information to come.

Trainwreck

Provides an interface to automate the formatting and sending of a message to the current chat room. Format is [#RekdTrain](link to random image of trainwreck). More information to come.

Main Site

Inline Editing Everywhere

Enables inline editing even if you don't have the inline editing privilege on a given site. Edits must still go through review if you don't have the editing privilege. More information to come.

Sticky Vote Controls

Makes voting controls stick to the top of the page when scrolling down a long post. More information to come.

Review

Magic Tag Review

Provides a new review interface for reviewing questions in a specific tag based on close or delete votes. A new section is provided one the /review page. This will continue to exist but is going to be superseded by the new Magic Review Tool which will have many more features. More information to come.

NATO Enhancements

Loads the full post and question as they would be seen in the wild with all controls enabled. More information to come.

NATO Hide Questions

Hides the questions loaded by the NATO Enhancements script, this will eventually be merged as an option into the NATO Enhancements script.

Review Enhancements

Loads all posts as they would be seen in the wild with all controls enabled while in review. More information to come.

Votes Page Enhancements

Loads posts as they would be seen in the wild with all controls enabled on your votes page. Useful for reviewing posts that you have voted on.

Thing Checker

A proof of concept audit detection script which intercepts the review AJAX requests and looks for the isAudit property of the returned JSON object. This is not intended to be used. If you use it, you run the risk of being manually review banned. A better option is to review to the best of your ability and report any bad audits on meta. A review ban incurred by multiple bad audits can be lifted by moderators if the bad audits are indeed bad audits.
